Prerequisitie Steps :
------------------------
Login Github : https://github.com/

Step 1: Create new account(signup) in github application
Step 2: create new repository in github. It will generate repsitory URL:
		https://github.com/arunrajput17/WebDriverManagerDemo.git
Step 3: install git plugin in eclipse
	Eclipse --> Help --> Eclipse Marketplace --> Search Git --> EGit - Git Integration for Eclipse (Install)

----------------------------
Step 1: Create / clone git repository in Eclipse
	- search option(Quick access --> Git Repository) --> clone a git repository --> provide URL
	- Local GIT rep location: D:\Day Shift\Selenium Practice\git\WebDriverManagerDemo

--> commit --> select all files in staged area, drag and drop--> press on commit and push

Step 3: Push our project files from GitHub to Git --> Eclipse
